Linux 交集/并集/差集，sed过滤列

June

技术/Linux

发布于：2019年2月28日

Linux 命令

两个文件比对，求交集，并集，差集。

a.txt
b.txt

方法1: sort

交集：

$sort a.txt b.txt | uniq -d
aaaaa
ccccc
eeeee
ggggg

并集：

sort a.txt b.txt | uniq
aaaaa
bbbbb
ccccc
ddddd
eeeee
fffff
ggggg
iiiii
jjjjj
kkkkk
lllll

差集：

sort a.txt b.txt b.txt | uniq -u
bbbbb
ddddd
fffff

方法2: comm

comm a.txt b.txt

显示三列：
第一列：a中有，b没有。a差b comm -2 -3 a.txt b.txt
第二列：a没有，b有。 b差a comm -1 -3 a.txt b.txt
第三列：ab都有 a交b comm -1 -2 a.txt b.txt

大小写互转

注意： 以下命令Mac OS无效

sed ‘s/[a-z]/\u&/g’
sed ‘s/[A-Z]/\l&/g’

拆解每一行中固定关键字的列

sed ‘s/.(action=[a-zA-Z]).(cmd=[_a-zA-Z-]*)./\1 \2/‘
等价于
sed ‘s/.(action=[a-zA-Z]{1,}).*(cmd=[_a-zA-Z-]{1,})./\1 \2/‘
加号不行
sed ‘s/.(action=[a-zA-Z]+).*(cmd=[_a-zA-Z-]+)./\1 \2/‘
